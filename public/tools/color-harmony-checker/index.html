What is the Color Harmony Checker?

The Color Harmony Checker by Abetworks is a free, powerful tool that generates beautiful color palettes based on color theory principles. Whether you're a designer, artist, or developer, create harmonious color schemes instantly using complementary, analogous, triadic, split-complementary, and tetradic harmony rules.

How to Use

  1. Use the color picker to select your base color, or enter a HEX code directly
  2. Alternatively, input RGB values (0-255) for precise color selection
  3. Choose a harmony type from the tabs (Complementary, Analogous, etc.)
  4. View the generated color palette with all harmony colors
  5. Click any color code to copy it, or download the entire palette as PNG

Color Harmony Types

🎨 Complementary

Two colors opposite each other on the color wheel. Creates maximum contrast and visual tension. Perfect for call-to-action buttons and highlighting important elements.

🌈 Analogous

Three colors adjacent to each other on the color wheel. Creates serene, comfortable designs with low contrast. Ideal for nature-inspired themes and cohesive branding.

🔺 Triadic

Three colors evenly spaced around the color wheel (120° apart). Offers strong visual contrast while maintaining balance. Great for vibrant, playful designs.

🎭 Split-Complementary

One base color plus two colors adjacent to its complement. Provides high contrast with less tension than complementary. Versatile for many design applications.

⬜ Tetradic (Rectangular)

Four colors arranged in two complementary pairs. Offers rich color schemes with multiple variations. Best used with one dominant color and others as accents.
