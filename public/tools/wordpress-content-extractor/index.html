What is the WordPress Content Extractor?

The WordPress Content Extractor by Abetworks is a free, powerful online tool that fetches recent posts from any WordPress website using the WordPress REST API. Extract post titles, dates, authors, excerpts, featured images, and categories. Perfect for content research, competitor analysis, and content auditing.

How to Use

  1. Enter the WordPress website URL
  2. Select the number of posts to fetch (5-100)
  3. Click "Fetch Posts" to retrieve content
  4. View post details including title, date, author, and excerpt
  5. Copy all posts or export as CSV for further analysis

Features

  • ✅ Fetch recent posts from any WordPress site
  • ✅ Get title, date, author, and excerpt
  • ✅ Extract featured images
  • ✅ View post categories and tags
  • ✅ Copy all posts to clipboard
