What Is My IP Address?

Whether you need to check your IPv4 or IPv6 address, find your ISP, or get geographic details, our tool provides comprehensive data with a single click.

Frequently Asked Questions

Is this IP lookup tool completely free?

Yes, this IP lookup tool is 100% free with no limits or registration required.

What is the difference between IPv4 and IPv6?

IPv4 uses 32-bit addresses (e.g., 192.168.1.1) while IPv6 uses 128-bit addresses (e.g., 2001:0db8:85a3::8a2e:0370:7334). IPv6 provides more addresses.

Does this tool store my IP address?

No, we do not store or log your IP address. All lookups are performed in real-time.
