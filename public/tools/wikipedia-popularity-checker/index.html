What is the Wikipedia Popularity Checker?

The Wikipedia Popularity Checker is a free analytics tool that lets you view and analyze page view statistics for any Wikipedia article. Using the official Wikimedia REST API, this tool provides real-time data on how many times an article has been viewed over the last 30, 60, or 90 days. Perfect for researchers, content creators, marketers, and anyone curious about topic popularity trends.

How to Use

  1. Enter the exact Wikipedia article title in the input field
  2. Click "Add Article" to fetch page view data
  3. Select your preferred time range (30, 60, or 90 days)
  4. View the interactive chart showing daily page views
  5. Add more articles to compare popularity trends
  6. Export your data as CSV for further analysis

Features

  • ✅ Real-time page view data from Wikimedia API
  • ✅ Flexible time ranges: 30, 60, or 90 days
  • ✅ Interactive bar chart visualization
  • ✅ Compare multiple articles side by side
  • ✅ Key statistics: total views, daily average, peak day
  • ✅ Trend indicator (up/down/stable)
  • ✅ Export data as CSV for further analysis
  • ✅ No signup required, completely free
