Convert Case

Transform your text between different cases instantly. Convert to uppercase, lowercase, title case, sentence case, and more with a single click.

About Convert Case Tool

Our free Convert Case tool allows you to quickly transform text between different case formats. Whether you're a developer, writer, or student, this tool helps you format text exactly how you need it.

Available Case Conversions:

  • ✓ UPPERCASE - All letters capital
  • ✓ lowercase - All letters small
  • ✓ Title Case - First Letter Of Each Word Capital
  • ✓ Sentence case - First letter of sentence capital
  • ✓ camelCase - first Word Lowercase, Rest Capitalized
  • ✓ PascalCase - All Words Capitalized
  • ✓ snake_case - lowercase_with_underscores
  • ✓ kebab-case - lowercase-with-dashes
  • ✓ CONSTANT_CASE - UPPERCASE_WITH_UNDERSCORES
  • ✓ AlTeRnAtInG - Alternating case
  • ✓ iNVERSE cASE - Invert current case
  • ✓ Capitalize First Letter - Of each word

Use Cases:

This tool is perfect for developers formatting code variables, writers fixing capitalization, social media managers creating eye-catching posts, and anyone who needs to quickly transform text case.
